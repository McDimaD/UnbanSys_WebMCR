Ручная установка

Внимание! Ручную установку стоит использовать только в том случае, если файл instruments/menu_items.php был изменен

В противном случае, воспользуйтесь автоматической установкой

Открываем instruments/menu_items.php и перед строчкой

'options' => array (

Добавляем

'unban' => array (

'name' => 'Платный разбан',
'url' => Rewrite::GetURL('unban'),
'parent_id' => -1,
'lvl' => 1,
'permission' => -1,
'active' => false,
'inner_html' => '',
),
  • Открываем configs/unban.cfg.php и настраиваем
  • Заливаем все файлы, кроме instruments/menu_items.php, в главный каталок webMCR
  • Переходим по адресу http://ваш_сайт.домен/go/unban
  • Готово
